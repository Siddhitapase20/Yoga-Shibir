Yoga Shibir Poster Editor

A simple web-based poster editor for creating Yoga Shibir posters with a live preview.  
Users can edit event details through input fields, see changes instantly on the poster image, and download the final poster as a PDF.

 Features

- Live poster preview while editing text
- Editable event details (Sthan, Dinank, Samay, Soujanya, etc.)
- Text rendered dynamically on a fixed poster image
- Download poster as a high-quality PDF
- Fully frontend-based (no backend required)

 Tech Stack

- HTML
- CSS
- JavaScript
- Sanscript.js
- jsPDF
